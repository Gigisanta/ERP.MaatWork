# Documentación del Proyecto

Esta carpeta contiene toda la documentación técnica y de arquitectura del proyecto CACTUS CRM.

## Estructura

- **`ARCHITECTURE.md`** - Arquitectura detallada y decisiones técnicas
- **`CACHING_STRATEGY.md`** - Estrategia de caché y optimización
- **`CRITICAL_COMPONENTS_AUDIT.md`** - Auditoría de componentes críticos
- **`IMPROVEMENTS-AUM-IMPORT.md`** - Mejoras al sistema de importación AUM
- **`MODULE_GUIDES.md`** - Guías de módulos y componentes
- **`N_PLUS_ONE_AUDIT.md`** - Auditoría de problemas N+1
- **`PROGRESS_SUMMARY.md`** - Resumen de progreso del proyecto
- **`REFACTORING_SUMMARY.md`** - Resumen de refactorizaciones y mejoras
- **`RUNBOOK.md`** - Runbook operativo del sistema
- **`TODOS_AUDIT.md`** - Auditoría de TODOs pendientes
- **`bloomberg.md`** - Documentación de integración Bloomberg
