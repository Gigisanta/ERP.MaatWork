Instrucciones para probar el error de login:

  1. Abre la aplicación en http://localhost:5173
  2. Ve a la página de login
  3. Ingresa cualquier usuario (ej: "test")
  4. Ingresa una contraseña incorrecta (ej: "wrong")
  5. Haz clic en "Iniciar Sesión"
  6. Verifica que aparezca el mensaje de error "Contraseña incorrecta" en la interfaz (no solo en consola)
  7. Abre las herramientas de desarrollador (F12) y ve a la pestaña Console
  8. Deberías ver los logs de debug que agregamos

Logs esperados en consola:

  • 🔵 LOGIN_FORM: Estado de error cambió: (vacío al inicio)
  • 📝 LOGIN_FORM: Iniciando submit
  • 🚀 LOGIN_FORM: Llamando función login...
  • ❌ LOGIN_FORM: Error en login: Error: Contraseña incorrecta
  • 🔴 LOGIN_FORM: Estableciendo error en estado: Contraseña incorrecta
  • 🔴 LOGIN_FORM: Error establecido, debería aparecer en UI
  • 🔵 LOGIN_FORM: Estado de error cambió: Contraseña incorrecta
  • 🟢 LOGIN_FORM: Renderizando error en UI: Contraseña incorrecta
